# Taiwan Avoid Moped Routes

This project uses Taiwan’s Department of Transportation data on vehicle frequency, road smoothness, and width to generate safer, more comfortable driving routes for larger vehicles, avoiding congested moped-heavy and impassable areas.

## Setup
Download `hotosm_twn_roads_lines_shp` from [https://data.humdata.org/dataset/hotosm_twn_roads](https://data.humdata.org/dataset/hotosm_twn_roads) and place it in the same folder as `Taiwan_AvoidMoped.py` (or a `Data` subfolder).
